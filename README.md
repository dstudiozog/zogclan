## 🚀 Deploy ke GitHub Pages

1. Upload seluruh folder `zog-clan-website/` ke repository GitHub
2. Buka **Settings → Pages**
3. Pilih branch `main` → folder `/ (root)`
4. Klik **Save** — website live dalam beberapa menit!

---

## 🖼️ Cara Tambah Foto Gallery

Tidak perlu edit kode apapun. Cukup:

### 1. Upload foto ke folder yang sesuai:

| Folder | URL Section |
|--------|-------------|
| `gallery/clan/` | Clan Gallery |
| `gallery/tournament/` | Tournament Gallery |
| `gallery/certificates/` | Certificates Gallery |

### 2. Nama file pakai angka:

```
0.jpg
1.png
2.webp
3.gif
...
100.jpg
```

### 3. Format yang didukung:
`jpg` · `jpeg` · `png` · `gif` · `webp` · `avif` · `bmp`

### ✅ Selesai! Website otomatis menampilkan foto.
